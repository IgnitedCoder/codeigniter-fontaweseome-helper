# codeigniter-fontaweseome-helper
A simple helper lib for font-aweseome

Pretty easy to use.

```
//Load it
$this->load->helper('faicons');
```
```
//use it
<h1>Here is an icon...</h1>
<?php echo fa_icon('ambulance') ?>
```
